**🔖 Project ID:** #CC5258  
**🎓 Internship Domain:** Java Development Intern (Entry Level)  
**🏢 Assigned By:** CodeClause Internship  
**🖥️ Environment:** Java • Eclipse IDE • OpenWeatherMap API

---

## 📌 Objective
Build a Java-based console application that allows users to check the **current weather** of any city by integrating with a public weather API.

---

## 🚀 Features
- 🏙️ Accepts **city name input** from the user  
- 🌐 Fetches **real-time weather data** from OpenWeatherMap  
- 📊 Displays:
  - 🌡️ Temperature
  - 💧 Humidity
  - ☁️ Weather condition (Clear, Rain, Cloudy, etc.)
- ⚠️ Includes basic error handling for:
  - Invalid city names
  - API/network errors

---

## 🛠️ Technologies Used
- Java (Core Java, OOP)  
- HTTP Requests using `HttpURLConnection`  
- JSON Parsing using `org.json` library  
- OpenWeatherMap API  
- Eclipse IDE  
- Maven (for dependency management)
